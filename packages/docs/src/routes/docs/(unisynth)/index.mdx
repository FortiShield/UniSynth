# Welcome to Unisynth

Unisynth is a new kind of web framework that can deliver instant loading web applications at any size or complexity. Your sites and apps can boot with about 1kb of JS (regardless of application complexity), and achieve consistent performance at scale.

## Why Unisynth?

<div class="card-grid">
  <div class="card">
    <h3>General-purpose</h3>
    <p>Unisynth can be used to build any type of website or application.</p>
  </div>
  <div class="card">
    <h3>Instant Interactivity</h3>
    <p>Unisynth apps work instantly without any delay because they don't need hydration, regardless of their size or complexity.</p>
  </div>
  <div class="card">
    <h3>JS Streaming</h3>
    <p>Unisynth delivers sub-second full page loads, even on mobile, by serving pure HTML and executing JavaScript when your users opt-in.</p>
  </div>
</div>
